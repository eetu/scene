# Credits

## Fonts (`frontend/static/fonts/`)

- **TopazPlus a1200** — Amiga system font used for the tracker/player surfaces.
  GPL with Font Exception. <https://github.com/rewtnull/amigafonts>
- **WebPlus IBM VGA 8x16** — exact CP437 VGA text font used for DOS-era NFO/DIZ
  art (box-drawing tiles seamlessly). From the *Ultimate Oldschool PC Font Pack*
  by VileR, licensed **CC BY-SA 4.0**. <https://int10h.org/oldschool-pc-fonts/>

## Audio

- **chiptune3 / libopenmpt** — module playback engine, vendored from the sibling
  `tracker` app. chiptune3 by DrSnuggles (MIT); libopenmpt (BSD).
